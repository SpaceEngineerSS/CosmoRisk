## First Launch

When you first open CosmoRisk, you'll see:

1. **Loading Screen** - App initialization
2. **Tutorial Modal** - 5-step onboarding (recommended for new users)
3. **Main Interface** - 3D solar system view

## Setting Up NASA API Key

1. Get your free API key at [api.nasa.gov](https://api.nasa.gov/)
2. In CosmoRisk, enter your key in the **API Key** input field (left panel)
3. Click **Fetch NEOs** to load real asteroid data

## Quick Start Workflow

### 1. Load Asteroids
- Enter NASA API key
- Click **Fetch NEOs**
- Wait for data to load

### 2. Explore
- Rotate and zoom the 3D view
- Click on asteroids to see details
- Check **Torino Scale** rating

### 3. Run Simulation
- Click **Play** button
- Adjust **Time Scale** (1x to 1,000,000x)
- Watch orbital mechanics in action

### 4. Analyze Threats

- View **Impact Analysis** panel
- Check probability and MOID

### 5. Plan Deflection
- Select target asteroid
- Choose deflection method
- Apply impulse and observe trajectory change
