# ⌨️ Keyboard Shortcuts

## Camera Controls

| Key | Action |
|-----|--------|
| `1` | Sun View |
| `2` | Earth View |
| `3` | Top-Down View |
| `4` | Ecliptic View |
| `R` | Reset Camera |
| `F` | Focus on Selected |

## Simulation Controls

| Key | Action |
|-----|--------|
| `Space` | Play/Pause |
| `+` / `=` | Increase Time Scale |
| `-` | Decrease Time Scale |
| `0` | Reset Time Scale |
| `Backspace` | Reset Simulation |

## Selection

| Key | Action |
|-----|--------|
| `Tab` | Cycle Through Asteroids |
| `Shift + Tab` | Cycle Backwards |
| `Escape` | Deselect All |
| `Ctrl + Click` | Multi-Select |

## Panels

| Key | Action |
|-----|--------|
| `H` | Toggle Help Overlay |
| `G` | Toggle Glossary |
| `T` | Toggle Tutorial |
| `C` | Open Comparison Table |

## Quick Actions

| Key | Action |
|-----|--------|
| `D` | Apply Deflection (if asteroid selected) |
| `I` | Toggle Impact Analysis |
| `O` | Toggle Orbits Visibility |
| `L` | Toggle Labels |

## Mouse Controls

| Action | Control |
|--------|---------|
| Rotate | Left Mouse + Drag |
| Pan | Right Mouse + Drag |
| Zoom | Scroll Wheel |
| Select | Left Click |
| Context Menu | Right Click on Object |

## Mobile/Touch Controls

| Action | Gesture |
|--------|---------|
| Rotate | One Finger Drag |
| Zoom | Pinch |
| Pan | Two Finger Drag |
| Select | Tap |
